# CLAUDE.md — South by AI

## What This Is

AI-powered SXSW 2026 schedule builder. Users take a quick quiz, get a personalized schedule generated by Claude, refine it via chat, and share it with a unique link.

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Key Files

- `docs/plans/2026-03-05-implementation-plan.md` — **THE PLAN. Read this first.**
- `data/sessions.json` — scraped SXSW session data (created by scraper agent)
- `docs/copy.md` — all copy (created by copywriter agent)
- `docs/brand.md` — brand guide (created by brand designer agent)
- `.env.local` — environment variables (KV credentials, etc.)

## Stack

- Next.js 14 (App Router)
- React 18 + TypeScript
- Tailwind CSS
- `@anthropic-ai/sdk` — Claude API
- `@vercel/kv` — Redis storage (Upstash)
- `@vercel/og` — OG image generation
- Vercel — hosting + deployment

## Design System

- Background: `#0A0A0A` | Primary: `#FF6B35` | Accent: `#00D4AA`
- Text: `#F5F5F5` | Muted: `#6B7280`
- Headings: Space Grotesk (bold) | Body: Inter
- Cards: glassmorphism (`bg-white/5 backdrop-blur-md border border-white/10`)

## Environment Variables

Already configured in `.env.local`:
- `KV_REST_API_URL` / `KV_REST_API_TOKEN` — Vercel KV (Upstash)
- `ANTHROPIC_API_KEY` — available in shell environment, add to `.env.local` if needed

## Auth

Stub auth for prototype — simple name input, no OAuth. Schedules keyed by random ID, not user accounts.

## Deploy

```bash
npx vercel --prod
```

Project already linked to Vercel. KV store already connected.

## Conventions

- TypeScript strict mode
- Tailwind for all styling (no CSS modules)
- Server components by default, `'use client'` only when needed
- API routes in `app/api/`
- Shared types in `lib/types.ts`
- Git commits after each logical unit of work
